The E–V–C Loop

Execution → Verification → Correction — the core algorithm

Algorithm 1: AutoSelf E–V–C Loop

Input: Plan P with dependencies; state s

t ← 0

while P ≠ ∅ do

t ← t+1

EXECUTE Et; observe Δt; update st

Propose Pt+1 (enabled tasks)

VERIFY issues ← Analyze(Pt+1, st)

while issues ≠ ∅ do

CORRECT Pt+1 ← Correct(Pt+1, issues)

issues ← Analyze(Pt+1, st)

end while

Et+1 ← Pt+1; dispatch; remove completed

end while
